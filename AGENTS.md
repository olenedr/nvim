# Agent Instructions

## Keybindings cheatsheet

`keybindings.html` is a browser-based searchable cheatsheet of all Neovim keybindings.

**Whenever you add, remove, or change a keybinding** — in `lua/keymaps.lua`, `lua/plugins.lua`, anywhere else in this config, or in `~/.tmux.conf` — you must also update `keybindings.html` to reflect the change before considering the task complete.
